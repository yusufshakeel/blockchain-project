## Getting started

#### Install packages

Run the following command to install the packages.

```shell
npm i
```

#### Bootstrap

This will create folder like `output` and wallets.

```shell
npm run bootstrap
```

#### Tests

To run the tests run the following in the terminal.

```shell
npm run test
```

## Start local server

Run the following in the terminal.

```shell
npm run start
```
